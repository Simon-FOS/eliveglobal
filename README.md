## About the Project

Elive Global is a construction and consultancy company. This web application allows the company to:

- Display services: consultancy, design & construction, renovation, project management  
- Showcase completed and ongoing projects with images and descriptions  
- Provide easy contact options for consultations and quotes  
- Highlight client testimonials  
- Present a professional and responsive interface for desktop and mobile users  

---

## Features

- **Static Pages**: Home, About, Services, Projects, Contact  
- **Media Integration**: 46+ project images uploaded in `public/img`  
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile  
- **Modular Express.js Structure**: Easy to maintain and scale  
- **Custom Styling**: Tailored CSS for a professional and modern look  

---

## Technologies

- **Backend**: Node.js, Express.js  
- **Frontend**: HTML5, CSS3, Bootstrap 5, Tailwind CSS  
- **Version Control**: Git, GitHub
